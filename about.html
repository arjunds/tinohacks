What is this App?

This app is meant to help recommend products a user should buy off eBay based on the environment in their house and previous purchases

What did we Use?

This app was made using an Intel Edison board as well as a variety of Grove sensors, namely the temperature, sound, and light ones. We used a Node.js backend server to send data to our frontend. Also, the trigger for our program was an Alexa command that we programmed. Most importantly, we used the eBay API as a service to buy the products.
Intel Edision used for the IoT part of our hack.
